INSTRUCCIONS DELS EQUIPS

Mode de creació d'equips
-------------------------------------------------------------------------------------
ARROWS - Changes current selection
PAG SEG / ANT - Chooses next or previous team

INTRO - Activa l'equip seŀlecionat
A - Activa / desactiva l'equip seŀlecionat
N - Edita el nom de l'equip

F1,F - Canvia el mode de pantalla sencera

R - Tria un equip a l'atzar / Inicia el mode joc

Mode joc
-------------------------------------------------------------------------------------
INTRO - Inicia el torn         
S - Mostra / amaga estadístiques

Mode presenació
-------------------------------------------------------------------------------------
Z, X - Puja o resta punts
1..6 - Activa o desactiva una categoria

F1,H - Ajuda | F2 - Crèdits | Q o ESC - Acaba
