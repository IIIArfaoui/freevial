TEAM INSTRUCTIONS

Team creation mode
-----------------
ARROWS - Changes current selection
PAG SEG / ANT - Chooses next or previous team

INTRO - Enables selected team
A - Enables / Disables selected team
N - Changes team name

R - Chooses a random team / start game mode

F1 - Changes full screen mode

Game mode
-----------------
INTRO - Starts turn

Presentation mode
-----------------
Z, X - Increment or decrease score
1..6 - Enables / Disables seleced category

F1,H - Help | F2 - About freevial | Q or ESC - Quit
